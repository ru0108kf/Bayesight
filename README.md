# Bayesight

## Getting Started

To get started, simply install the required dependencies using pip.

```bash
python -m venv Bayesight

# Windows
Bayesight\Scripts\activate
# macOS/Linux
source Bayesight/bin/activate

pip install requirements.txt
```
